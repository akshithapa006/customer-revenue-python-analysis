Customer Revenue Analysis (Python)

Tools Used :
* Python
* Pandas
* Matplotlib

Description :
This project analyzes customer transaction data using Python to understand revenue trends and customer spending patterns.

Key Analysis :
* Total spending per customer
* Identification of top customers
* Revenue distribution by city
* Data visualization using bar charts

Files :
* `analysis.py` – Python code for data analysis
* `customer_revenue_analysis.xlsx` – Dataset used for analysis

Conclusion :
The project demonstrates how Python can be used for data analysis and visualization to extract meaningful business insights.
